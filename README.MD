# Linux Screen Time Widget

Minimal Conky widget that displays your screen time for the current session.

## Preview

![Demo](demo.gif)

## Features

- Lightweight
- Simple bash implementation
- Clean Conky widget
- Easy install

## Installation

```bash
git clone https://github.com/yourname/screen-time-widget.git
cd screen-time-widget
chmod +x install.sh
./install.sh
```

Run the widget:

```bash
conky -c ~/.screen_time_widget/screen_time.conkyrc
```
